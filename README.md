## pin assignments

|pin|assignment|
|:--|:--|
|GP0|out (74hc165 SL)|
|GP1|out (74hc165 SCK)|
|GP2|in  (74hc165 MISO)|
|GP3|in  (send switch)|
|GP4|in  (uart RX)|
|GP5|out (uart TX)|
